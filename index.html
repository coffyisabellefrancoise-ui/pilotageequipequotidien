🔥 Configurer Firebase

Pour partager les données avec vos collègues, configurez Firebase gratuitement en 5 minutes. Suivez ces étapes :

1
Allez sur console.firebase.google.com et créez un projet (ex: pilotage-equipe)
2
Dans le projet, activez Realtime Database → Créer une base → Mode test
3
Allez dans Paramètres du projet → Vos applications → Web et copiez les informations de configuration
4
Collez les valeurs ci-dessous et cliquez Connecter
💡 La base de données en mode test est gratuite et accessible à tous sans compte. Vos collègues n'ont rien à configurer, ils voient les données automatiquement.
